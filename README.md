# AR_glasses_controller Gesture Recognition

Recognizing Hand-Gestures using OpenCV and Python to controll the phone on AR_glass.

**Libraries needed**
* cv2
* imutils
* numpy
* sklearn

**Usage**
* python segment.py
* python recognize.py
* python recognize-image.py

**BlueTooth-mouse simulating**
* mouse_client.py
* mouse_emulate.py
